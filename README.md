# UQ Encyclopedia Dashboard

Interactive dashboard for the Uncertainty Quantification Encyclopedia benchmark results.

**View the live dashboard**: This branch contains the GitHub Pages deployment.

## What's Here

This branch contains the interactive HTML dashboard and all visualization files for browsing UQ benchmark results:

- **Linear Models**: Bayesian regression, Hat Matrix, Conformal Prediction
- **Nonlinear Models**: Nlinfit, Bayesian nonlinear, ensemble methods
- **Data-Driven Models**: Gaussian Processes, Neural Network ensembles

## Repository

For the full source code, benchmark scripts, and documentation, see the [main branch](https://github.com/jkitchin/uq-encyclopedia/tree/main).

## About

This dashboard presents comprehensive benchmark results for various uncertainty quantification methods applied to regression tasks. All results include both interpolation and extrapolation evaluation.

See [COVERAGE_EXPLAINED.md](https://github.com/jkitchin/uq-encyclopedia/blob/main/COVERAGE_EXPLAINED.md) on the main branch for details on how metrics are computed.
